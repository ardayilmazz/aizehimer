# AI-Zheimer - Akıllı Hatırlatıcı Asistan

## Proje Hakkında
AI-Zheimer, yapay zeka destekli bir hatırlatıcı ve asistan web uygulamasıdır. Google Gemini API kullanılarak geliştirilmiştir.

## Teknolojiler
- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (Vanilla JS - şimdilik)
- React JS (ileride)
- Google Gemini API (ileride)

## Özellikler
- 🤖 AI destekli chatbot
- 💊 İlaç hatırlatıcıları
- 📅 Günlük/Haftalık/Aylık/Yıllık hatırlatıcılar
- 📝 Not alma
- 💾 Local Storage ile veri saklama

**Not — AI sohbet:** AI chat kısmı tamamen çalışacak şekilde hazırlandı fakat üyelik ödemesi yapılmadığı için kullanılamıyor.

## Kurulum
1. Projeyi klonlayın veya indirin
2. `index.html` dosyasını bir web tarayıcısında açın
3. Tailwind CSS CDN üzerinden yüklenecektir (internet bağlantısı gerekli)
